# 🌤 Weather App

This is a simple and responsive Weather Application that provides real-time weather information using a Weather API.

---

## 🚀 Features
- 🌍 Get real-time weather data of any city
- 🌡 Shows temperature, humidity, and weather condition
- 🔍 Search functionality by city name
- 📱 Fully responsive design

---

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Weather API
